Tlk

The LAN Kommunicator

Fork On GitHub
Downloads:

    TLK pronounced as talk, is a LAN communication software. The aim of this project is to provide easy chatting and file transfer utility for Linux and Windows.

    Features version 0.1:

    • Automatic peer discovery.

    • Basic chat functionality.

    Upcoming features:

    • File transfer support.

    • System tray functionality.

    • Windows version.
